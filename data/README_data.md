# Data description

This project uses UK Biobank participant-level data and derived dietary flavonoid data. Raw UK Biobank data are not included in this repository because they are subject to UK Biobank access restrictions and participant confidentiality requirements.

Place the following files in `data/raw/` before running the pipeline:

- `participants.recall.over1.csv`
- `participants.recall.over2.csv`
- `demographics.csv`
- `4907incidence.csv`
- `484895control.csv`
- `average_24hrecall_individual_foods.csv`
- `flavonoid_ukbiobank.csv`

The analysis can be reproduced by approved researchers after obtaining access to UK Biobank and preparing files with the variable names described in the manuscript.
